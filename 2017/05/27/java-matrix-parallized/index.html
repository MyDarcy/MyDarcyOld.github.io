| 字数统计

矩阵运算在很多地方都会用到，常见如图像，密码学。这里来学习一下矩阵算法的典型实现–矩阵乘法的并行化实现。

矩阵乘法原理

  • 矩阵乘法中，第一个矩阵的列数和第二个矩阵的行数目必定相等，如果需要进行并行计算，一个简单的策略就是将A矩阵进行水平分割，得到子矩阵A1和A2, B矩阵进行垂直分割，得到子矩阵B1和B2, 这个时候我们只需要分别计算这些子矩阵的乘积，然后将结果进行拼接，就能得到原始矩阵A和B的乘积。当然，子矩阵也是可以进一步进行划分的。
  • 使用ForkJoin框架来实现这个并行矩阵相乘的算法。同时使用 jMatrices 作为矩阵计算的工具。
